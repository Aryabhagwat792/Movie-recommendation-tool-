PROBLEM STATEMENT
There are a lot of movies available so it becomes difficult to find one movie to watch especially it becomes difficult to find a particular genre . Generally we rely on random suggestions and we waste a lot of time searching or deciding which one to watch. This project aims at that and gives recommendations based on genre instantly.

Scope of the project
The project focuses only on content based recommendations using prompts given
Taking name from user matching the text 
Giving all suggestions
This uses small dataset and one python code so its convenient

Target users
Anyone who requires quick movie suggestions
People who prefer offline recommendation
Those who want to understand how recommendation system works

High level features
List of movies can be edited can be edited
Runs on any system with python
